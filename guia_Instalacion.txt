Para instalar CGAL se requiere instalar antes Cmake, Boost, GMP, MPFR
Comandos:
	*Instalar Cmake
	 sudo apt-get install Cmake
	 Para revisar la version de cmake actual se usa
	 cmake --version

	*Instalar Boost
	 sudo apt install libboost-dev

	*Instalar GMP
	 sudo apt-get install m4

	*Instalar MPFR
	 sudo apt install libmpfr-dev

	*Instalar CGAL
	 sudo apt-get install libcgal-dev

	*Instalar Qt
	sudo apt-get install qt5-default

	
	*Agregar los examples de CGAL para correrlos luego
	sudo apt-get install g++ cmake libcgal-demo
	tar --gzip -xf /usr/share/doc/libcgal-demo/examples.tar.gz

	*Ir a la direccion
	cd examples/

	*Crear el correcto CMakeLists para CGAL
	cgal_create_CMakeLists
	*Dentro de un ejemplo se corre el siguiente comando
	cmake -DCMAKE_BUILD_TYPE=Release -DCGAL_USE_BASIC_VIEWER=ON .
	make

	*Luego correr ejecutable
	./ejecutable

	Agregar la libreria Faltantes:
	Ambas librerias estan en la carpeta incl

	Agregar la libreria faltante draw:	draw_voronoi_diagram_2.h
	https://fossies.org/linux/CGAL/include/CGAL/draw_voronoi_diagram_2.h
	Lo agrego a mi path en /usr/include/CGAL pero puede variar
	sudo cp -rfp incl/draw_voronoi_diagram_2.h /usr/include/CGAL/

	Agregar la libreria faltante de Qt: Basic_viewer_qt.h
	https://fossies.org/linux/CGAL/include/CGAL/Qt/Basic_viewer_qt.h
	sudo cp -rfp incl/Basic_viewer_qt.h /usr/include/CGAL/Qt/

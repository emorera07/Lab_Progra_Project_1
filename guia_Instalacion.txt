Para instalar CGAL se requiere instalar antes Cmake, Boost, GMP, MPFR
Comandos:
	*Instalar Cmake
	 sudo apt-get install Cmake
	 Para revisar la version de cmake actual se usa
	 cmake --version

	*Instalar Boost
	 sudo apt install libboost-dev
	 
	*Instalar GMP
	 sudo apt-get install m4

	*Instalar 
	 sudo apt install libmpfr-dev

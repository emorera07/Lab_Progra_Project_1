Para instalar CGAL se requiere instalar antes Cmake, Boost, GMP, MPFR
Comandos:
	*Instalar Cmake
	 sudo apt-get install Cmake
	 Para revisar la version de cmake actual se usa
	 cmake --version

	*Instalar Boost
	 sudo apt install libboost-dev
	 
	*Instalar GMP
	 sudo apt-get install m4

	*Instalar MPFR
	 sudo apt install libmpfr-dev

	*Instalar CGAL
	 sudo apt-get install libcgal-dev


	*Agregar los examples de CGAL para correrlos luego
	sudo apt-get install g++ cmake libcgal-demo
	tar --gzip -xf /usr/share/doc/libcgal-demo/examples.tar.gz

	*Ir a la direccion
	cd examples/
	
	*Dentro de un ejemplo se corre el siguiente comando
	cmake -DCMAKE_BUILD_TYPE=Release .
	make

	*Luego correr ejecutable
	./ejecutable
